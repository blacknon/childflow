# Code of Conduct

This project aims to be welcoming, respectful, and constructive.

## Expected Behavior

- Be kind and assume good intent.
- Focus feedback on ideas, code, and behavior.
- Share technical disagreement without hostility.
- Help keep issues and pull requests useful for future readers.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language
- Repeated bad-faith arguing or derailing discussions
- Publishing private information without permission
- Spam or deliberately disruptive behavior

## Scope

This applies to project discussions, issues, pull requests, and review conversations.

## Reporting

If something needs attention, open a private security-style report by email if available, or open a neutral issue asking for maintainer follow-up without escalating the conflict in public.
